# sd
了解REST：
	1.每个URI代表一种资源
	2.每中资源用唯一的URI，其表现形式的不同可以在HTTP请求中指定
	3.URI中不包含动词，动词表现在HTTP协议中
	4.HTTP四种动词：GET  POST PUT DELETE

综上，使用Spring MVC框架处理请求，并返回Json格式的数据，实现前后台分离

---------------------------
大体结构：
controller层：中间层，得到url后调用其他层，返回结果

dao层：数据库查询

entity层：数据库表对应的类

service层：实现具体业务处理

content层：数据的再包装

---------------------------
根据员工id及月份等到具有 员工ID 员工姓名 员工上下班时间 员工工作总时长 员工是否迟到早退，信息的表
---------------------------
查询方法：根据员工ID写一个sql语句在多张表中查询，直接返回结果OR多次查询，在后台部分组装成考勤表。
-------------------------------
签到表设计：员工id，年，月，日，时间  5个字段组成主键，确定唯一的一条信息
